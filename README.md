# TaskManager

Este projeto consiste em um sistema de gerenciamento de tarefas desenvolvido em Java, com o objetivo de praticar e consolidar conceitos fundamentais de programação backend. A aplicação permite criar, listar, buscar, concluir e remover tarefas por meio de uma interação simples via terminal.

Durante o desenvolvimento, foram utilizados conceitos de Programação Orientada a Objetos (POO), como encapsulamento e separação de responsabilidades, além do uso de estruturas de dados como ArrayList para armazenar e manipular as tarefas em memória.

O projeto também explora a organização de código em diferentes camadas, separando a lógica da aplicação da representação dos dados, aproximando a estrutura de aplicações reais utilizadas no mercado.

## Funcionalidades
- Criação de tarefas
- Listagem de tarefas
- Busca por ID
- Marcação de tarefas como concluídas
- Remoção de tarefas

## Tecnologias utilizadas
- Java
- Programação Orientada a Objetos (POO)
- Estruturas de dados (ArrayList)

## Melhorias futuras
